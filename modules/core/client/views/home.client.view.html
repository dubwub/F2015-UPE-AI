AICOMP.IO


RPI & UPE's very first AI programming competition! Over $5000 in prizes!

What is this?

Inspired by existing AI competitions at other schools (i.e. MIT’s Battlecode and Pokerbots), UPE has created an AI competition for RPI students. This is motivated by the same factors that motivate UPE’s semesterly ProgComp: UPE is always looking for new talent, and UPE loves to foster interest in coding at RPI. Plus, there’s cash and sponsors involved, what a plus!

In case you’re not familiar with AI competitions in general, here’s a general overview of how this one will work.

UPE has devised and programmed a game that takes AI input and uses those to determine a winner -- that’s where you come in! The full rules of the game, as well as the source code of the game engine are all released at the beginning of the competition. With these as a guide, RPI students in teams of four or less will spend the semester developing and testing their AIs to most optimally play the game.

UPE will be running a server with the game logic on it continuously throughout the semester that teams can use to test their AIs. This server will handle matchmaking and have an internal ELO system that will be used to rank AIs before the main events.

At the main event, all interested teams will battle to the death in a large double elimination bracket, and a champion will be crowned.

The game is a modified variant of Bomberman, which can best be described as a deterministic Bomberman with portals. The full description of everything that is modified is explained in explicit detail in the documentation.

How do I get involved?

  • If you're not familiar with Bomberman, rectify that immediately! It's a classic retro game that you may have played when you were a kid. The rules for this competition are somewhat different from traditional Bomberman, so go to Games > Training Mode to play around with it for yourself or read up in the Documentation!
  • Create an account from the Sign Up page, and get started making your AI! The details of testing your AI against the server are in the Documentation!
  • Have fun!

Learn

  • Read the Bomberman documentation
  • Play around with the game yourself

Develop

Code in your favorite language! The server and clients communicate using HTTP requests (see the documentation for more details), and pretty much all languages can interface this way. Go nuts!

Test

Testing your AI against other AIs on the matchmaking server is super easy using our API! Check out the Documentation for more details.

Compete

More on this coming later! There will be a seeding tournament and a main event that's open to entry to all interested RPI teams.
