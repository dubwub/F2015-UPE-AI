UPE AI COMPETITION


Deterministic Bomberman with portals!

What is this?

This is an AI competition where we provide the game and the rules, and you guys provide the AIs that compete for prizes!

How do I get involved?

  • If you're not familiar with Bomberman, rectify that immediately! It's a classic retro game that you may have played when you were a kid. The rules for this competition are somewhat different from traditional Bomberman, so go to Games > Training Mode to play around with it for yourself or read up in the Documentation!
  • Create an account from the Sign Up page, and get started making your AI! More information on competing and testing your AI will come later!
  • Have fun!

Learn

  • Read the Bomberman documentation
  • Play around with the game yourself

Develop

Code in your favorite language! The server and clients communicate using HTTP requests (see the documentation for more details), and pretty much all languages can interface this way, so go nuts!

Test

More on this coming later! There will be a way to connect to the server with matchmaking details to test your AI online against other searching AIs.

Compete

More on this coming later! There will be a seeding tournament and a main event that's open to entry to all interested RPI teams.
